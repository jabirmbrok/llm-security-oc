# LLM-SOC: Hybrid Alert Triage Dashboard

LLM-SOC is a proof-of-concept Security Operations Center dashboard for alert triage and response guidance. The system is designed to combine classical machine learning predictions with Qwen/LoRA-based LLM explanations through an external API. The Streamlit interface supports single-alert analysis, dataset exploration, mock LLM mode, and future integration with a cloud VM inference server.

## Current Status

- Streamlit dashboard implemented
- Mock LLM response mode enabled for development
- External Qwen/LoRA API client prepared
- Dataset explorer and API debug page included
- Future integration planned with trained Qwen LoRA model on VM
